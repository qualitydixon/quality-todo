# quality-todo

## In this project...

- React
- Redux
- Webpack
- LESS
- ESLint (standard)
- React Hot Loader

## Installation

Just fork and clone this repo, run `npm install`, then `npm run start`. Then go to `http://localhost:8080/` in your browser.
